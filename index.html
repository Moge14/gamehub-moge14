Dev Log
v0.5.0
March 15, 2026
Feature
Echoes of Vel'Keth — New Game Launch
A second original title is now live on GameHub. Echoes of Vel'Keth is a narrative walking experience set on a lush alien world at golden-hour sunset. Explore crystal valleys, glowing pools, bioluminescent mushrooms, and ancient stone altars as you piece together the final memories of a vanished civilisation.
  • Full 3D outdoor world with real-time blue sky, sun disc, and volumetric clouds
  • Predictive frustum culling — objects stream in before you look at them, freed behind you
  • Async chunk build queue with 5ms frame budget — geometry spread across frames, no stutter
  • Detailed bioluminescent flora: mushrooms with gills, rim glow, and spots; multi-blade grass with wind sway
  • 7 narrative echo fragments to discover across 3 named zones
  • Free-mouse look mode (pointer lock where available, graceful fallback in iframes)
v0.4.0
March 13, 2026
Feature
Friends System & Account Improvements
A full friends system is now live. Add anyone on the platform by username, send and receive friend requests, and view their profile, game library, and playtime.
  • Add friends by username with live cross-account request state
  • Accept, decline, and cancel friend requests
  • Friend profile side-drawer showing games and playtime
  • Pending requests badge on the Friends nav button
v0.3.0
March 13, 2026
Feature
Accounts, Local Storage & Playtime Tracking
GameHub now has a full local account system. Credentials are stored using PBKDF2 hashing via the browser's native SubtleCrypto API. Playtime is tracked per game.
  • Account creation with PBKDF2-hashed passwords (100k iterations)
  • Session persistence — stay logged in across page reloads
  • Personal library with playtime tracking per title
  • Password strength meter on registration
v0.1.0
March 13, 2026
Launch
GameHub & Stellar Abyss — Initial Launch
First public release of GameHub alongside Stellar Abyss — a first-person 3D space shooter built in Three.js r128, running entirely in the browser.
  • Xbox-inspired launcher UI with Home, Game Pass, Library pages
  • Stellar Abyss: three explorable areas, NPC dialogue, full FPS controls
  • Game Pass page with trending and new releases sections
